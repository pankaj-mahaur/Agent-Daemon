# Examples

Copy-paste configuration templates for Claude Code. Adapt to your project.

| File | Description |
|------|-------------|
| [settings-global.json](settings-global.json) | Global `~/.claude/settings.json` — model, plugins, marketplace |
| [settings-project.json](settings-project.json) | Project `.claude/settings.json` — hooks, permissions |
| [hooks-graphify.json](hooks-graphify.json) | PreToolUse hook for graph-aware code exploration |
| [CLAUDE.md.example](CLAUDE.md.example) | Template for project documentation |
| [AGENTS.md.example](AGENTS.md.example) | Template for multi-repo workspaces |
